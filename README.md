# 🏥 HEALPER 智慧健康管家 (Smart Health Assistant)

> 國立中央大學 資訊管理學系 畢業專題 
> **🏆 核心亮點：成功串接 LLM 與穿戴裝置數據，提升系統回覆效率達 40%**

## 📖 專案簡介 (About The Project)
HEALPER 是一款專為優化健康資訊查詢與管理而開發的 Android 智慧健康應用程式。本專案結合了原生 Android 開發技術、Fitbit 健康數據追蹤，以及大型語言模型 (LLM)，透過智慧化的對話介面，提供使用者即時、準確且白話的健康諮詢服務。

## ✨ 核心功能與成效 (Key Features & Achievements)
* **🤖 智慧健康客服 (LLM Integration)：** 成功串接 OpenAI ChatGPT API，實作上下文記憶與特定 Prompt 限制，確保回覆內容符合專業健康管家的人設。
* **🏃‍♂️ 健康數據同步 (Wearable Integration)：** 整合 Fitbit API，即時抓取與分析使用者健康數據，提供客製化建議。
* **⚡ 效能與體驗優化：** 透過優化 API 請求流程與非同步處理，大幅降低等待延遲，**實測提升回覆效率達 40%**。
* **☁️ 雲端與後端服務：** 導入 Firebase 進行後端資料同步與使用者管理。

## 🛠️ 技術堆疊 (Tech Stack)
* **前端 / App 開發：** Kotlin, Android Studio
* **API 串接：** OpenAI API(gpt-3.5), Fitbit API, RESTful API
* **後端 / 雲端服務：** Firebase
* **版本控制：** Git, GitHub
